## Featured Project: Raspberry Pi 5 Homelab

A production-grade home server running **24 containerized services** on a single Raspberry Pi 5 (8 GB), managed entirely through Docker Compose.

**Key technologies:** Docker Compose, Cloudflare Zero Trust, Pi-hole DNS, Telegram alerting, Restic + Backblaze B2 backups.

**Services by category:**

| Category | Count | Highlights |
|----------|------:|------------|
| Media and Storage | 6 | Immich, Jellyfin, Paperless-ngx, Syncthing |
| Monitoring | 8 | Beszel, Uptime Kuma, LoggiFly, Speedtest Tracker |
| Networking and Security | 2 | Pi-hole, Cloudflared tunnel |
| Management | 3 | Portainer, Homepage dashboard, Backrest |
| Automation | 2 | Home Assistant, N8N |
| Utilities | 3 | Warracker, iSponsorBlockTV, Kaneo |

**Hardware:**

| Component | Specification |
|-----------|---------------|
| Server | Raspberry Pi 5, 8 GB RAM |
| OS Storage | 64 GB SD Card |
| Data Storage | 1 TB External SSD (USB 3.0) |
| Network | Gigabit Ethernet |
| OS | Raspberry Pi OS (Debian 12 Bookworm, 64-bit) |
